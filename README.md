# Vet-React

### Features
- [x] Criação do cliente
- [x] Leitura do cliente
- [x] Edição do cliente
- [x] Remoção do cliente
- [x] localStorage para armazenar os clientes

### Inicialização

Para realizar a instalação dos pacotes:
```
npm install
```

Rodar aplicação:
```
npm run dev
```

### Ferramentas Utilizadas

- Vite
- React
